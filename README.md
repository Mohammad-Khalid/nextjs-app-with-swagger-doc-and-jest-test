# Starter Demo NextJS App

This is a starter demo NextJS App which demonstrate how to add Jest test and swagger docs to the NextJS App.

## Getting Started

Install NPM packages

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Swagger page and config

- To see swagger doc open [http://localhost:3000/api-doc](http://localhost:3000/api-docs)
- You can start editing the swagger doc configuration by modifying `pages/api-doc.js`

## Jest test

All the Jest test file can be found in `/tests` directory

Run Jest Test:

```bash
npm run test
```
